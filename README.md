# Monorepo template for Micro-frontend with shared Vue3/2 components

# Development

- Install dependencies:
```sh
  pnpm install
```

- Run Vue 3 dev server:
```sh
pnpm dev:vue3
```

- Run Vue 2 dev server:
```sh
pnpm dev:vue2
```
